<h1 align="center">🏢 Student Admin System</h1>
<h5 align="center">基于Java 17、Gson、Gradle、Lambda、Hutool开发的Cli的学生管理系统</h5>

------

# 如何使用
> 所需环境：  
> Java 17+  
> 良好的网络

1. 构建项目
``` bash
git clone https://github.com/Core2002/StudentAdminSystem.git  
cd StudentAdminSystem
gradlew shadowJar
```

2. 运行
``` bash
java -jar build/libs/StudentAdminSystem-1.0-SNAPSHOT-all.jar
```
